1️⃣ What is the difference between var, let, and const?
 Ans: var: var is function-scoped. We can re-assign and re-declare, but it has a hoisting issue.

      let: let is block-scoped. We can re-assign but cannot re-declare in the same scope.

      const:const is block-scoped. We cannot re-assign and cannot re-declare.

2️⃣ What is the spread operator (...)?
      Ans: The spread operator (...) is used to expand an array, object, or iterable into individual elements. 

3️⃣ What is the difference between map(), filter(), and forEach()?
      Ans: map() creates a new array by appling a function to each element of the original array.
      filter() Creates a new array with elements that pass a condition.
      forEach() Executes a function on each element of the array.
